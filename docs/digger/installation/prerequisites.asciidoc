== Prerequisites

A Digger server can be installed using the https://github.com/aerogear/digger-installer[Digger Installer tool].

Before running the Digger installer, please make sure the following prerequisites are met:

=== Ansible

Ansible is installed locally. The required version is 2.2 and higher.  
If you have Ansible installed locally, you can check the Ansible version by running `ansible --version`.
Otherwise please check http://docs.ansible.com/ansible/intro_installation.html[Ansible installation instructions].

=== Java

Please make sure Java is installed locally.

=== OpenShift

* A running OpenShift cluster running, and its command line tool (https://github.com/openshift/origin/releases/tag/v1.3.1[oc]) installed locally. The minimum required version is OpenShift 3.0.
* The OpenShift cluster should have at least 2 persistent volumes (PVs) with 10G capacity or more.
