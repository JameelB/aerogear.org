AeroGear iOS

2.0.0 brings major changes:

  • modularization: AeroGear-iOS repo is splitted in several smaller libs.
  • deprecated Pipe: http calls use lean aerogear-ios-http directly built on top of NSURLSession for accessing RESTful services.
  • enhance OAuth2 module with support for Google, Facebook, Keycloak and a pluggable OAuth2 module.
  • and Swift based
