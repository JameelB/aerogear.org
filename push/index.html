AeroGear Push

AeroGear has support for different push notification technologies. The SimplePush Server represents an implementation of Mozilla's SimplePush specification, while the WebPush Server is an implementation of the WebPush Protocol specification.

The UnifiedPush Server itself offers a unified way to deliver push notifications to multiple push networks, such as Android, iOS or Windows.

If you have any questions, feel free to hit us up on the mailing list or on irc.
